TrueType Font (TTF)

Originally designed by Apple, TrueType is used by both Apple and Microsoft in their Mac OS and Windows operating systems. Supported by all major browsers, it useful for extending compatibility to older browser versions, especially on mobile.

Web Open Font Format (WOFF/WOFF2)

Now you might be asking yourself, if TrueType has such high compatibility, why should I use any other formats?

Allow me to introduce the Web Open Font Format (WOFF). Developed by Mozilla specifically for use on the web, WOFF fonts are compressed versions of TTF or OTF that load significantly faster. It also allows for the inclusion of metadata and license information within the file.

Developed by Google, WOFF2 is the next generation of WOFF. The improved compression of WOFF 2.0 offers a 30% average gain over WOFF 1.0. However, it is only compatible with modern browser versions.

Efficiency and Compatibility

If your web app is mainly targeting users on modern browsers, you might be able to get away with just WOFF or WOFF2.

To expand support to as many legacy browsers as possible, you can use @font-face on CSS to serve multiple fallback font formats in case of failure. EasyFonts was created to help you do this in just a few clicks!


  @font-face {
    font-family: 'FontName';
    src: url('webfont.woff2') format('woff2'), /* Extremely modern browsers */
         url('webfont.woff') format('woff'), /* Modern browsers */
         url('webfont.ttf') format('truetype'); /* Safari, Android, iOS */
  }
